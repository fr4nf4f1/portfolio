Sistema de etiquetas para una línea de vinos inspirada en las constelaciones. El proyecto explora la deconstrucción tipográfica y el uso de gradientes para generar profundidad, creando una estética contemporánea y disruptiva dentro del marco vitivinícola. Cada constelación —Orión, Casiopea, Lyra— tiene su propia paleta y personalidad visual, unidas por un sistema tipográfico común.
